TRATAMENTO DE DADOS

A LGPD estabelece que os titulares possuem direitos sobre os dados pessoais tratados, a contratada possui meios de comunicação com o titular para que essas solicitações sejam recebidas, analisadas e atendidas, sendo o resultado positivo ou negativo.

Pela natureza de nosso negócio é possível que exista o compartilhamento de dados pessoais com a nossa empresa durante o estabelecimento de contratos de prestação de serviço com os clientes.

Com isso, a contratada segue o papel de operador dos dados de acordo com o que for determinado pelo cliente que assume o papel de controlador dos dados pessoais.

Quando houver o compartilhamento pela contratada com terceiros, este será realizado se for absolutamente necessário para o alcance dos resultados para os quais eles são mantidos e quando existirem arranjos contratuais e mecanismos de segurança apropriados.
